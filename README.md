# ReverseProxies

本项目集合了各种反向代理方案，可反代大部分的网站/api，如:

- GitHub
- Google
- OpenAI API
- And more...

Try now!

## 注意事项

无法反代一些需要登录查看内容的网站(**无法进行登录**)

大部分方案有用量限制(基本使用足够)

部分方案有一点问题，但基本不影响使用

## 使用

选择下面的一个方案，进入文档部署吧!

- [Cloudflare Workers](/doc/cfworker.md)
- [Vercel](/doc/vercel.md)
